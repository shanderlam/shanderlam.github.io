<p>
If you want to line up with equals sign in a series of assignment statements in emacs, you can use the align-regexp comment. In the minibuffer, type <kbd>Mx align-regexp RET = RET</kbd>. Below is a example.
</p>
<p>
Before:
<pre>
$a = 1;
$ab = 2;
$abc = 3;
</pre>
</p>
<p>
After:
<pre>
$a   = 1;
$ab  = 2;
$abc = 3;
</pre>
</p>
